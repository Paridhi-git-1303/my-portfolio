# 🌐 My First Personal Portfolio Website

Welcome to the repository for my very first personal portfolio website! This project serves as a digital home to showcase my skills, projects, and growth as a web developer. It features a fully custom design with smooth interactions and responsive layouts.

## 🛠️ Built With

This website was engineered entirely from scratch using fundamental web technologies:

- **HTML5:** Structuring clean, accessible web components and semantic page content.
- **CSS3:** Implementing unique styling, layout grids, and interactive visual feedback.
- **JavaScript (ES6):** Powering dynamic client-side mechanics and user event actions.

---

## 🌟 Key Features

- **Responsive Layout:** Automatically scales seamlessly for mobile screens, tablets, and desktops.
- **Modern UI Canvas:** Clean, minimalistic aesthetic built directly using customized vanilla CSS layers.
- **Smooth Navigation:** Easy-to-use menu allowing fast access to personal sections and content.
- **Project Grid:** Highlights interactive visual cards detailing my technical development journey.

---

## 📂 Project Structure

The codebase follows a simple, fast-loading, single-page architecture:
- `index.html` — The main structural document housing your text copy, layout, and image references.
- `styles2.css` — Custom stylesheets managing font treatments, colors, positioning, and animations.
- `script.js` — Client-side scripts controlling interactive behaviors and menu transformations.

## ⚙️ How to Run Locally

You can preview and test this portfolio locally on your machine without installing external framework dependencies.

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate into the project directory:**
   ```bash
   cd my-portfolio
   ```

3. **Launch the project:**
   - Double-click the `index.html` file to launch it directly inside any default web browser.
   - Alternatively, open the directory in VS Code and hit **"Go Live"** using the Live Server extension.
